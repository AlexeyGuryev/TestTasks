Постановка задачи

Перед Вами стоит задача - разработать web приложение для моделирования расположения мебели в комнатах.
Программа должна предоставлять возможность (API):
Создавать, удалять комнаты;
Создавать, удалять предметы мебели определенного типа;
Перемещать предметы мебели между комнатами;
Хранить историю перемещений мебели между комнатами;
Получать информацию о состоянии комнат, и наличия предметов мебели в них на любой момент времени в истории.


Описание предметной области

Существует некоторое (произвольное) количество комнат, в каждой из которых может находиться мебель.
Предметы мебели имеют определенный тип (стул, стол и т.д.). 
Конкретный предмет мебели не может находиться в двух комнатах одновременно или ни в какой из комнат, т.е., когда мебель создается, она сразу должна быть помещена в комнату.
Предметы мебели можно перемещать между комнатами в любом порядке (можно перенести все предметы в одну комнату, оставить какую-либо комнату абсолютно пустой и т.д.).


Требования к реализации

Язык программирования - C#

Хранение данных на ваше усмотрение
UI на ваше усмотрение

Описание методов API

Нише описан абстрактный API, как именно он будет реализован остается на ваше усмотрение, главное чтобы приложение позволяло выполнять данные команды.
create-room -room <название комнаты> -date <дата>
Создаёт комнату на указанную дату, выводит название созданной комнаты. Имя комнаты должно быть уникально.
remove-room -room <название комнаты> -transfer <название комнаты куда мигрирует мебель> -date <дата>
Удаляет комнату на указанную дату и мигрирует мебель из удалённой комнаты в указанную.
create-furniture -type <тип мебели> -room <название комнаты> -date<дата>
Создаёт тип мебели на указанную дату с указанным названием в указанной комнате.
Если в комнате уже есть заданный тип мебели, увеличивается количество экземпляров данного типа в комнате.
Если данного типа нет в комнате - создается новый тип и в комнате с количеством - 1.
move-furniture -type <тип мебели> -from <название комнаты откуда перемещать> -to <название комнаты куда перемещать> -date<дата>
Перемещает мебель указанного типа из комнаты from в комнату to на указанную дату.
query [-date <дата>]
Вывод списка комнат и количеств каждого типа мебели внутри них.
Если дата не задано, то берётся текущее состояние.
history [-short]
Вывод полной истории состояний комнат, включая дату.
Если задан параметр -short, выводим только даты.

Примечание:
Все методы API быть непротиворечивы по отношению к датам.
Например, нельзя удалить комнату с датой удаления меньшей даты создания комнаты.
